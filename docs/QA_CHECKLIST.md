# QA Checklist

## Functional QA
- [ ] Home, services, contact, and thank-you routes render.
- [ ] Netlify form submissions appear in dashboard.
- [ ] Required validation appears for empty form submissions.

## SEO QA
- [ ] All indexable pages have unique title/meta.
- [ ] Canonical tags resolve to intended URLs.
- [ ] Sitemap and robots are accessible.
- [ ] Redirect map implemented and tested.

## Performance QA
- [ ] Lighthouse scores meet target thresholds.
- [ ] No heavy runtime bundles introduced.
- [ ] Non-critical images use lazy loading.

## Cross-Browser QA
- [ ] Chrome latest
- [ ] Safari latest
- [ ] Firefox latest
- [ ] iOS Safari and Android Chrome spot checks

## Launch Day
- [ ] DNS and TLS verified.
- [ ] Form submission test complete.
- [ ] Analytics events visible.
- [ ] Rollback plan prepared.
