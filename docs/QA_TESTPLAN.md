# QA Test Plan

## Functional
- Verify all core routes load.
- Verify both forms submit and redirect.
- Verify sticky call button behavior on mobile viewport.

## SEO
- Verify title/meta/canonical per route.
- Verify schema blocks on home/service/FAQ pages.
- Verify robots and sitemap endpoints.

## Conversion
- Validate CTA visibility above fold on homepage.
- Confirm form_start and submit events emit.
- Confirm call_click events emit from header/footer/sticky.

## Regression
- Compare old/new heading parity and key service copy.
- Validate redirect behavior for legacy URLs.
