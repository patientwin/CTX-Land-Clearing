# SEO Strategy

## Keyword Clusters by Page Intent
- Home: `land clearing austin`, `forestry mulching austin`
- Land clearing page: `land clearing austin tx`, `lot clearing austin`
- Forestry page: `forestry mulching austin tx`, `brush mulching austin`
- Contact: `land clearing estimate austin`, `forestry mulching quote`
- Service areas: `land clearing near austin`, `central texas land clearing`

## On-Page Strategy
- One H1 per page with service + location alignment.
- Preserve old semantic intent while tightening title/meta quality.
- Keep internal links between home, services, and contact in every funnel path.

## Local SEO Elements
- NAP consistency in header/footer/contact.
- LocalBusiness schema on homepage.
- Service schema on service-intent pages.

## Schema Plan
- Home: `LocalBusiness`
- Service pages: `Service`
- FAQ sections: `FAQPage` where FAQ copy exists
- Breadcrumbs: `BreadcrumbList` on inner pages

## Indexing and Crawl Considerations
- Static HTML output for predictable crawlability.
- `robots.txt` allows crawl and references sitemap.
- Redirect map included for URL changes.
